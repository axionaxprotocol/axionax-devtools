# Axionax Devtools

Scripts and utilities that support Axionax protocol development.

## Contents

- `scripts/` - Automation for local nodes, tooling, and CI helpers
- `tools/` - Benchmark and maintenance utilities

## Usage

Most scripts are POSIX shell compatible. Run with `bash` or `sh` and read inline comments before execution.

## Contributing

1. Fork this repository and create a feature branch.
2. Document any new scripts and keep them idempotent.
3. Add usage examples to script headers when possible.
4. Open a pull request describing the purpose of the tooling.
